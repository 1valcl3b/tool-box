executar dentro da vm Pnetlab, para acessar a interace web do zabbix pelo navegador do HOST

iptables -t nat -A PREROUTING -p tcp --dport 8080 -j DNAT --to-destination IP-DO-ZABBIX:80
iptables -t nat -A POSTROUTING -j MASQUERADE

acessar no navegador:

IP-DO-HOST:8080

Exemplo: http://10.0.75.71:8080
